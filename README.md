# Neulbom

Neulbom Care — 보호자와 돌봄자를 연결하는 AI 기반 공유 일지 앱

## 개발 환경

- Node.js 18 LTS 이상 권장
- npm 9 이상 (Expo CLI 포함)
- Expo Go 앱 (실기기 테스트 시)

## 프로젝트 시작하기

```bash
npm install
npx expo start
```

## 환경 변수 설정

Supabase 연동을 위해 루트 디렉터리에 `.env` 파일을 생성하여 다음 값을 입력하세요.

```bash
cp .env.example .env
```

```env
EXPO_PUBLIC_SUPABASE_URL=your-supabase-url
EXPO_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
```

## 주요 기능

- **피보호자 관리**
  - 돌봄자가 피보호자를 등록·수정·삭제하고, 연결된 보호자를 선택할 수 있습니다.
  - 보호자와 돌봄자는 여러 피보호자를 관리할 수 있으며, 메인 화면 상단에서 대상을 전환해 활동일지를 확인합니다.

- **활동일지**
  - 돌봄자가 활동일지를 작성하며 사진 업로드, 식단/태그 기록, AI 노트 활용이 가능합니다.
  - 작성된 활동일지는 수정·삭제할 수 있고, 삭제 시 연결된 댓글도 함께 정리됩니다.

- **댓글 소통**
  - 보호자와 돌봄자 모두 활동일지에 댓글을 남길 수 있으며, 자신의 댓글은 수정·삭제할 수 있습니다.
  - 댓글은 실시간으로 새로고침해 최신 대화를 확인할 수 있습니다.

- **권한 기반 뷰**
  - 로그인한 계정의 역할(보호자/돌봄자)에 따라 버튼 노출과 기능이 자동으로 조정됩니다.

## 화면 흐름

1. **로그인/회원가입**  
   이메일 기반 인증으로 보호자/돌봄자 계정을 생성합니다.
2. **메인 화면**  
   피보호자를 선택해 활동일지 목록을 확인하고, 돌봄자는 새 일지를 작성하거나 피보호자를 관리합니다.
3. **활동일지 상세**  
   상세 내용을 확인하고, 돌봄자는 수정/삭제, 보호자·돌봄자는 댓글을 통해 소통합니다.
